## üé• Production Workflow

### 1. Preview Animation
```bash
# Open any HTML file in browser
# Press F11 for fullscreen
# Click anywhere to restart
```

### 2. Screen Record
- Set recording to **1080x1920** (vertical)
- Or let browser scale automatically
- Record full 22-28 second loop
- Tools: OBS, QuickTime, Windows Game Bar

### 3. Create Voiceover
- Open corresponding `-VOICEOVER.txt` file
- Copy script into ElevenLabs
- Use recommended voice & settings
- Download as MP3 (44.1kHz)

### 4. Edit
- Import screen recording + audio into CapCut/Premiere
- Sync audio to video using timestamps
- Optional: Add subtle background music (keep VERY quiet)
- Export vertical 9:16 for TikTok

### 5. Post to TikTok
- Use provided hashtags from voiceover file
- Add caption (examples in voiceover files)
- Post at optimal times (12pm, 7pm, 9pm)
